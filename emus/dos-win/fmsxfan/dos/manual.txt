公式ＨＰにアップされているfmsxfan.lzh以外は保証外です。



紹介：
＝＝＝
このプログラムはＭＳＸエミュレータです。
このソフトは他のＭＳＸエミュレータとは違い、ゲームを遊ぶためではなく
ＭＳＸソフトを開発できるエミュレータを作る事を目的として開発されています。

現段階ではＭＳＸ２＋までの完全再現を目標としています。
それが終わった場合、turboRのエミュレーションに移行する予定です。


確認した問題点：
＝＝＝＝＝＝＝＝
	グラフサウルス２でマウスのクリックができない。
	雑音が入るときがある。


必要環境：
＝＝＝＝＝
	PentiumIIくらいのCPUと、DOS上で640x480x16が表示できて速いビデオカード
	(Celeron366MHz,Xpert98とAudioPCIで、WinだとOK、DOSだと遅い）


インストール方法：
＝＝＝＝＝＝＝＝＝
アーカイブを解凍して、システムROMイメージを同じディレクトリに入れて下さい。
このプログラムはCWSDPMI.EXEというDPMIサーバを使用するので、CWSDPMI.EXEを
持っていない場合はそれも一緒にホームページからダウンロードして同じディレクトリ
に解凍してください。（解凍するだけ。DOS窓で使用する場合は必要ない）

なお、ROMイメージを作成するツールは付属していませんので自力でどうにかして下さい。
吸い出した機種によっては対応出来ないものがあるかもしれません。

対応しているROMイメージは、
DISK.ROM
FMPAC.ROM
KANJI.ROM
KNJDRV.ROM(32kのもの)
MSX.ROM
MSX2.ROM
MSX2EXT.ROM
MSX2P.ROM
MSX2PEXT.ROM
MSXDOS2.ROM
PAINTER.ROM
RS232.ROM
です。（コンフィグファイルで名前を変更できます。）

特徴：
＝＝＝
GUI対応。
コンフィグファイル対応。
ディスクドライブはディスクイメージにのみ対応。物理ドライブは使用できません。
ディスクの交換はA､Bドライブに３２枚ずつ。
OPLL,PSG,SCC,1ビットPCM（不完全）サポート。
スクリーン１０以降、MSX-DOS2、漢字ROM、漢字BASIC対応。
マウス一部対応。
インターレース対応（SCREEN5,7のみ）
ジョイスティックは４ボタン以上の場合、ボタン３と４で連射ができます。
サウンドのwave録音。

＝＝＝＝＝＝＝＝＝＝＝

F6		・・・SPACEキー連打のON/OFF

F8		・・・ドライブAに登録したディスクイメージの切り替え

Home		・・・Home
End		・・・SELECT
Pause		・・・Pause
R Ctrl		・・・Ctrl、ジョイパッドのボタン２
L Ctrl		・・・Graph
L Alt		・・・ジョイパッドのボタン１
PageUp		・・・STOP
PageDown	・・・かな(Country)
テンキーのEnter	・・・テンキーの','


オプション：
＝＝＝＝＝＝
括弧内はデフォルト。

-rom、-joyについては、スロット２、ポートＢを指定するときは
fmsxfan xxx.rom zzz.rom -rom 2 -rom 3 -joy 1 -joy 2 のように指定してください。

ディスクイメージについては、
fmsxfan -diska a.dsk b.dsk -diskb c.dsk d.dsk のように指定します。

  -verbose ?    - Select debugging messages [1]
                        0 - Silent       1 - Startup messages
                        2 - V9938 ops    4 - Disk/Tape
                        8 - Memory      16 - Illegal Z80 ops 
  -hperiod <period>   - Number of CPU cycles per HBlank [228]
  -vperiod <period>   - Number of CPU cycles per VBlank [59736]

  -pal/-ntsc          - Set PAL/NTSC HBlank/VBlank periods [NTSC]

  -printer filename - Redirect printer output to file [stdout]

                        (multiple -diska options accepted)
  -diskb <filename>   - Set disk image used for drive B: [DRIVEB.DSK]
                        (multiple -diskb options accepted)
  -tape filename    - Set tape image file [off]
  -font filename    - Set fixed font for text modes [DEFAULT.FNT]
  -auto/-noauto     - Use/Don't use autofire on SPACE [off]

  -vram pages       - Number of 16kB VRAM pages [2/8/8]
  -rom type         - Select MegaROM mapper types [0,0]
                        (two -rom options can be present)
