* 변경 내용

  파일뷰어의 정렬 모드가 늘었습니다.
  CTRL+S 키로 모드 변경이 가능합니다.
  [정렬 안함] > [이름순] > [확장자+이름순]으로 바뀝니다.
  기본 정렬 모드는 M.INI에 설정가능합니다.

  FDD의 가상 드라이브 액세스 시, 멈추는 문제가 수정되었습니다.


* File manager - M

  M은 MS-DOS에서 널리 쓰이던 Mdir과 비슷한 프로그램입니다.
  파일 관리 및 실행을 도와줍니다.
  인터레이스 고해상도 모드를 사용하며 완성형 한글도 지원합니다.
  넌인터레이스 모드 사용 시에는 8x8 폰트로 표시됩니다.
  (내장 폰트 대신, 외부 파일 로딩 가능. 2048바이트 폰트)


* 요구 사항 - 표준 버전
  MSX-DOS2가 구동되는 MSX2(VRAM 128KB) 이상의 기종.
  추가 메모리 매퍼 192KB 사용.


* 요구 사항 - LITE 버전
  MSX-DOS2가 구동되는 MSX2(VRAM 128KB) 이상의 기종.
  추가 메모리 매퍼 128KB 사용.
  #IMSP 및 인터레이스 모드가 삭제된 버전입니다.


* 요구 사항 - R800_TR 버전
  내부적으로 R800 명령을 쓰므로, A1ST/GT에서만 구동가능합니다.
  속도 이득외에는 표준 버전과 동일합니다.


* 설치
  PATH가 걸린 디렉토리에 압축파일을 모두 풀어넣습니다.
  만약 LITE 또는 R800_TR 버전을 쓰려면 해당 디렉토리 내의
   M.COM, M.OVL 두개 파일을 교체하면 됩니다.


* 사용법

  PATH 걸린 디렉토리에 설치가 되었으므로,
  DOS2 프롬프트에서 M [리턴] 하면 실행됩니다.

  파일 뷰어 상에서의 동작키에 대한 설명은,
   M 실행 후 F1을 눌러 #HELP를 실행하면 볼 수 있습니다.


* 환경 설정 - M.INI

  사용자 환경 설정은 M.INI 파일로 저장됩니다.
  펑션키 또는 파일 확장자별 실행 프로그램을 선택할 수 있습니다.
  사용법은 M.INI 파일을 참고하세요.

  확장자에 따른 글자색(16진수 사용)을 지정할 수 있으며,
   배경색도 바꿀 수 있습니다.
  파일의 기본색은 회색(E)이며, 배경색은 파란색(BG:4)입니다.

  실행 프로그램 설정시 %s 부분은 현재 선택된 파일/디렉토리로
  바뀐 후 실행됩니다.

  확장자별 실행 프로그램은 세미콜론을 사용하여,
    여러개(최대 8개)의 프로그램을 등록할 수 있습니다.
  예) TXT:5:HV %s;APED %s
      MP3:9:#MPXP %s;MPXP %s

  기본적으로 TSR(램상주) 옵션이 켜져 있습니다.
  TSR 옵션을 해제하려면, <OPT> 그룹에서 TSR:0으로 설정하세요.
  이미 램상주 후에는, DOS상에서 M /r 를 입력하면 해제됩니다.


* 환경 설정 - _M_.BAT

  M에서 외부 프로그램을 실행 할때는
  _M_.BAT 파일을 이용하도록 되어있습니다.
  프로그램 실행 후의 PAUSE를 원치 않는 경우는,
  _M_.BAT 파일의 PAUSE 명령을 삭제하세요.

  _M_.BAT를 통한 외부 프로그램 실행 후 M으로 복귀 시,
  이전 파일뷰어의 디렉토리가 다시 표시됩니다. (TSR 옵션 필수)
  만약 TSR 옵션 OFF 일때는, 현재 디렉토리가 나타납니다.


* 임시 파일 생성

  LZH 압축 파일 처리시 _M$.BAT 파일이 임시로 생성됩니다.
  정상적인 동작 후 삭제되니 참고하세요.


* 명령 접두사

  파일명 앞에 *를 붙이면,
      M이 점유중인 메모리가 해제된 후 프로그램 실행됩니다.
      (메모리 부족한 환경에서 유용함.)

  디렉토리명 앞에 ]를 붙이면,
      해당 디렉토리로 이동 후, 프로그램이 실행됩니다.
  

* 내장 프로그램

  #MPXP: 김준성님의 MPX를 직접 구동하는 플레이어입니다.
         CPU 모드는 Z80 3.58MHz 및 R800 터보 지원합니다.
         프로그램 설정시 #MPXP를 쓰면 실행가능합니다.
         화면 하단에 MP3/WMA 진행바가 표시되며,
         MP3 ID3 태그 및 WMA info 태그를 지원합니다.
         ESC 또는 / 키를 눌러 MP3 중지를 할수 있습니다.

  #IMSP: MoonSound 를 이용한 IMS 음악 플레이어입니다.
         Z80 3.58MHz 모드에서는 느려질수 있습니다.
         LITE 버전에서는 지원되지 않습니다.
