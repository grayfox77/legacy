Introduction.-  

Eat Blue! was a game I created in 1995 for my MSX1, only in Basic, with very few graphics. It was more a survival game than and puzzle game and it was a 16k long single program. You can download the old version soon, from my web page.
For the MsxDev'04 I resolved to remake it, but with a puzzle style. So, It was recoded from zero, it's entirely new and more simply in concept. Now it's shorter (to fit in a 8ks ROM) and has ASM routines to speed up the game. Eat Blue! 2004 is here. Enjoy it!

''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

The Story.-

Since a lot time ago, Blue Eaters are surviving by eating blue squares.
The love blue squares! But they are difficult to find and, in some cases, they are in dangerous zones.
But our Blue Eater is hungry, so he must find some blue squares to eat. Help him!

''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

Game Control.-

You can use keyboard Stick or Joystick in port 1 to move through the menus and to move the character, and Space to select options from menu.

''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

Menus.-

Start: To start the game.

Continue: To enter a password and continue from upper levels. When you select this, you must type a 4 number password. You'll see it at the bottom part of the screen.

User: This is to load user levels, made with an external editor. You can load it from Cassette or Disk. If you type a letter from A to Z, it will try to load the file "USER_[letter_typed].EBL", which must contain Eat Blue! levels. If there is no disk, no disk unit, no EBL file, etc... it will return to main menu.
If you type another key, it will load from cassette, just "USER.EBL".

WARNING!! --- YOU MUST PRESS CTRL KEY AT STARTUP TO BE ABLE TO LOAD USER LEVELS ---

A simply level editor, as well as some levels, will be available on my web page, in a future.

''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''''

Game Basics.-

As you can see, Blue Eater is hungry and must eat blue squares or he die.
So, the objective on each level is to arrive to the exit, to go next level, before your energy or time goes to zero.

For this, it's usually necessary to eat the blue squares and avoid the red ones.
Dark blue squares have more food than normal blue ones and you can use it twice. Dark red squares cause more damage than normal red ones. Green squares have no effect and can't be eated (no good taste for Blue Eaters), and black ones are void space, so you can't pass through them.

Anyway, each movement you do it's causes loss of energy, so thing well your movements to save energy.

Before get the exit, you must have the correct key to open it. There are three keys on the level, but only one is the correct. Of course, if you get all three keys on the level,  you will open the exit for sure, but maybe you will not have enougth energy!

One more object is in the level, an Egg. If you get three eggs, you will get the current level password (there is a password for each five levels group). If you die, you will loss an egg, all the keys, and a live, of course.

That's all, so let's go for the blue!

El Juego.-

Como puedes ver, Blue Eater esta hambriento y debe comer casillas azules o se morira.
Por eso, el objetivo de cada nivel es llegar a la salida, para ir al siguiente nivel, antes de que tu energia se termine.

Para ello, suele ser necesario comer casillas azules y evitar las rojas.
Las de color rojo azul tiene mas comida que las normales y pueden ser usadas dos veces. Las de color rojo oscuro causan mas daño que las rojas normales. Las verdes no tienen efecto y no pueden ser comidas (no tienen buen sabor), y las negras son espacio vacio sobre el cual no puedes pasar.

Ademas, cada movimiento causa una perdida de energia, asi que piensa bien tus movimientos.

Antes de ir a la salida, debes tener la llave correcta para poder abrirla. Hay tres en cada nivel, pero solo una es la buena. Por supuesto, si consigues las tres, abriras la salida, pero quizas no tengas suficiente energia luego!

Otro objeto que podras ver en el nivel es un Huevo. Si consigues tres, podras ver el password del nivel actual (hay uno para cada grupo de cinco niveles). Si mueres, perderas un huevo, todas las llaves, y una vida, claro.

Eso es todo, a por el azul!
