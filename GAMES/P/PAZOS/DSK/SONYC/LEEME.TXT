				   SONYC

	Este fichero contiene informacion de ultima hora que no ha podido ser   incluida en el manual del juego. Lee detenidamente el contenido de este texto   antes de ejecutar el programa.


	1. Para ordenadores a 7Mhz
---------------------------------- 
	SONYC esta optimizado para ordenadores a 7Mhz. Si se ejecuta el juego a 7Mhz la musica no sonara correctamente. En este caso pulsa la tecla [SELECT]    durante la carga del juego hasta que aparezca el mensaje "Loading R800 version".De esta forma se usaran los controladores de sonido adecuados para este tipo de ordenadores.


	2. Para usuarios de MSX2+
---------------------------------
	Si el scroll no es suave durante el desarrollo del juego, es            recomendable desactivar los efectos sonoros y/o la musica del juego a traves    del menu de opciones. De esta forma el movimiento de la pantalla sera totalmentesuave. No obstante el juego es perfectamente jugable en ambos casos.

	3. Cambio de frecuencia
-------------------------------
	Este juego esta desarrollado para 50Hz, pero si por algun motivo tu
televisor o monitor funciona a 60Hz, puedes cambiar la frecuencia pulsando las
teclas 5 o 6 en el menu principal, para elegir 50 o 60Hz respectivamente. Aun
asi, se recomienda el uso de los 50Hz.

	4. Aviso contra la pirateria
------------------------------------
	Copiando este u otros programas originales solo estas contribuyendo a ladesaparicion del sistema MSX. Compra software original solo a distribuidores
autorizados.
