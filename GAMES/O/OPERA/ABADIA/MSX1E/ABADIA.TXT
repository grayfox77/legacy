Disk SAVE & LOAD
----------------

How to save to disk:

- CTRL+F1 Save data to memory
- CTRL+TAB Stops the game and enter disk SAVE/LOAD mode
- "S" Save data from memory to disk


How to load from disk:

- SHIFT+F1 Load data from memory
- CTRL+TAB Stops the game and enter disk SAVE/LOAD mode
- "L" Load data from disk to memory
