■SCREEN2 アニメーションプロジェクト（？）
	SCREEN2 というあまり使われないモードを使って、フル画面アニメー
	ションを実行しよう・・・という、無謀かつ無意味なプロジェクトで
	す (^_^;

■このソフト
	専用アニメーションデータ AS2形式（Animation for Screen2）を再生
	するツールです。データは HDD や RAMDISK等の高速なメディアへ転送
	して再生してください。遅いメディアでは動きがぎこちなくなります。

■動作
	MSX-DOS または MSX-DOS2 を起動して、コマンドラインから

	A>AS2PLAY BASKET.AS2

	のように実行すると再生を開始します。BASKET.AS2 はアニメーション
	ファイル名です。再生対象にあわせて適当なファイル名を指定してくだ
	さい。

■推奨環境
	次の環境を推奨します。

	MSXturboR + ATA-IDE + 遅くないLBAモードで動作するHDD (^_^;

	遅くない HDD とは、１ＧＢ以上のものならまず大丈夫でしょう。
	ノート用の 2.5インチHDD でも大丈夫です。

	外部RAMによる RAM-DISK で試したところ、上の推奨環境よりも遅くなり
	ました。数フレームごとに一瞬固まります。上の推奨環境ではなめらかで
	す。
