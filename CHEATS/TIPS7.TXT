  EL  programa  nos  permite  el  poder  juagar  a  5   juegos

    1 2 3 4 0
    2 3 4 0 1
    3 4 0 1 2
    4 0 1 2 3

    OPCIONES
    REDEFINIR PANTALLA (2 menu principal)

  Si optamos por este apartado del menu, pulsando espacio, nos
  saldra un menu con el siguiente contenido:

            FILE #0 (Pantallas y elementos  del juego 01234)
            FILE #1 (Pantallas y elementos  del juego 12340)
            FILE #2 (Pantallas y elementos  del juego 23401)
            FILE #3 (Pantallas y elementos  del juego 34012)
            FILE #4 (Pantallas y elementos  del juego 40123)

  Deberemos elegir uno de los  archivos  para  crear  nuestros
  propios elementos del juego

  Una vez seleccionado un fichero, nos aparecera el  siguiente
  menu:
  - SONIDO
  - VELOCIDAD
  - PARPADEO
  - ELEMENTOS DE DECORADO
  - REDIFICION DE SPRITES (16X16)
  - REDIFICION DE SPRITES (64X64)
  - EDICION DE PANTALLAS DEL 1-18
  - ACCION
  - SALVAR SALIR

  Si deseamos salir  del  menu,  sin  salvar  las
  modificaciones, pulsaremos BS

  explicaciones:
  SONIDO: podemos elegir entre el numero 0 al 4
  VELOCIDAD: podemos elegir entre el numero 0-1
  PARPADEO; podemos elegir entre el numero 0 al 39

  En edicion:  disponemos  de  las  siguientes  teclas
  estandard
  ESC: volver al menu principal
  ESPACIO:  confirmar
  B/N: menu de seleccion de colores

  EDICION DE PERSONAJES (menu 3)

  Eligiendo la 3 opcion del menu principal entramos en el menu
  de edicion de protagonista y opciones de los  programas.  En
  este caso, este es VALIDO para los 5  juegos  opcionales  de
  Construccion & Games.

  Veremos dos recuadros, en forma de matriz, en una  de  ellos
  se nos mostraran los personajes ya creados,  y  en  la  otra
  matriz podremos modificar/crear nuevos personajes.

  Principales opciones:

  ESC: salva modificaciones y regresa al menu principal
  BS: regresa al menu principal, sin salvar las modificaciones
  ESPACIO: confirmar
  V: colorea un pixel de color negro
  B/N: selecion de colores. Pulsando  espacio  se  colorea  el
  pixel.
  1- Cancela la figura de la matriz A/0, borrando  tambien  la
  matriz de edicion de 16*16
  2-  Colorea  toda  la  matriz  de  edicion  con   el   color
  seleccionado

    3-  Rota imagen
    4-  rota 180 gr. la imagen (espejo)
    5- Sube un pixel la imagen
    6- Baja un pixel la imagen
    7- La imagen avanza una posicion hacia la derecha
    8- La imagen avanza una posicion hacia la izquierda
    A- Animacion de la imagen seleccionada

  Atencion a los valores prefijados por el programa
  en la letra A: encontramos el personaje
              B y C: elementos de disparo
              D: elementos de explosion
              E: bonus

  Tambien con este programa podemos dotar a nuestros  enemigos
  de   movimientos   ofensivos,    capacidad    de    defensa,
  velocidad...,  Esperamos  que  este  breve  resumen  de   su
  funcionamiento os anime a realizar nuevos  juegos  con  este
  programa.
