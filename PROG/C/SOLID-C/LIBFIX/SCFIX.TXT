SOLID C Library bug fix package
By Tatsuhiko Syoji(Tatsu) 1999-2000

[本パッケージについて]
　本パッケージはSOLiD C librariesのバグを取ったものです。
　サポート担当のEgor Voznessenski氏にはメールで報告済みですが、リリース
される気配がないので私が発表します。
　この為、SOLiD C librariesのパッケージに含まれるライブラリのバグが直っ
た時点で本パッケージは使命を終えます。

[構成ファイル]
*       .ASM :バグを取ったファイルのソースファイルです。
MKRLIB  .BAT :RLIB.IRLを生成するバッチファイルです。
STDIO   .H   :バグを取ったstdio.hです。
RLIB    .IRL :バグを取ったライブラリです。
SCFIX   .TXT :本ファイルです。

[本ファイルの使い方]
　セットアップされているSOLID Cのヘッダファイルのあるドライブ・ディレク
トリにstdio.hを、ライブラリファイルのあるドライブ・ディレクトリに
rlib.irlをそれぞれコピーして下さい。
　プログラムをコンパイルする際は、リンカldでclib.irlの前にrlib.irlをライ
ブラリとしてリンクするようにして下さい。
例)test.relにclib.irlをリンクして実行ファイルを作成する
ld test,rlib/s,clib/s/gXMAIN

[ライセンス]
　本ライブラリはSOLiD C librariesのバグを取ったものであり、SOLiD C 
librariesのコードを含んでおります。
　SOLiD C librariesはSOLIDさんの著作物であり、SOLIDさんのライセンスが適
用されます。
　この為、私のバグ修正部分にも同じライセンスを適用します。
　SOLIDさんのライセンスは次のようになっております。

　以下の文は私がメールでサポート担当のEgor Voznessenski氏に問い合わせた
際の回答です。
SOLID C is freeware, and you may use it for any purpose.
You may use it for development of ANY software (including
commercial products). SOLID C cannot be sold.

　正式なものではありませんが、私の訳です。
SOLID Cはフリーソフトウェア、あらゆる用途に利用できます。
あなたは(商用ソフトを含めた)あらゆるソフトウェアの開発にSOLID Cを利用で
きます。
SOLID C本体は売ることはできません。
