       > Channel undependant
       Key	Description
       --------+-----------------------------------------------------------
       Cursors	Move through pattern
       BS	Clear step
       Del	Delete one step (move rest of channel backwards)
       Ins	Insert one step (move rest of channel forewards)
       Return	Write last item
       Home	Goto bottem/top of channel
       Select	Toggle video frequency (50Hz/60Hz)
       Stop	Stop playing. If view, Edit cursor at last played pattern
       F1	Play song from current position (view on)
       F2	Goto Original_User_Instrument editor
       F3	Goto FM_User_Drum_editor
       F4	Goto instrument_list
       F5	Open file

       Cursors	Change songlenght

       C	Copy channel(s)
       G	Goto channel
       +	Half note up (current step!)
       -	Half note down ,,      ,,

       F4	Goto channel_routing
