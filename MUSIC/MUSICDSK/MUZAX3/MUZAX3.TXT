How to install MUZAX #3 on your HardDisk.

1 Create a directory (e.g. C:\MUZAX3)
2 Copy the following files to this dir.:
  - MUZAX3.BIN  (The BASIC boot)
  - MUZAX3.BAT  (The DOS boot)
  - MUZAX3.BAS  (Sub file for DOS boot)
  - MUZAX3.LIB  (The main file)
