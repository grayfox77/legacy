Snow - XL2S Entertainment - 2003 - www.xl2s.tk

Coding by GhostWriter P.

Snow is an interactive snowfall demo! So try out the
following keys and see what happens:

	Left  	- Turn Windflow counter clockwise
	Right 	- Turn Windflow clockwise
	ESC/RET	- Exit to dos

The current Windflow direction is shown in the lower
right corner on the screen. Enjoy!
