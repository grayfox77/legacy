    - MRC "Bounce" Challenge -

This demo is optimized to run on PAL
MSX 1 machines. However, this demo
should work on every MSX with at
least 64Kb RAM.

The demo requires MSX-DOS to run. To
run the demo you should copy the
files in this archive together with
MSXDOS.SYS and COMMAND.COM on a
blank disk.  Start the demo by
typing BLEND when booted.
