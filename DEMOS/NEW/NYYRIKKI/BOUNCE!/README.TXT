
Bounce!
-------

Loading in MSX-BASIC:

WIDTH 40:KEY OFF:COPY "BOUNCE!" TO "CON"

Loading on MSX-DOS:

MODE 40
COPY BOUNCE! CON

This is not a program, so you can view this on any VT-52 compatible computer from phones to mainframe computers. This demo is anyway optimized for non Japanese 3.5MHz MSX1 with diskdrive.
