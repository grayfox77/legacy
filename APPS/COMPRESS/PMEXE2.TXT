Door  gebruik  te  maken van  de combinatie  PMARC.COM en  PMEXE2.COM is  het 
mogelijk om een gecrunchte file te maken die zichzelf kan uitpakken en meteen 
wordt uitgevoerd. Hierbij wordt de file NIET eerst op disk gezet, maar in het 
geheugen geplaatst, waardoor het erg snel gaat.

Hoe werkt het:

A>PMARC B:SHOW-EXE.COM=PMEXE2.COM B:SHOWEM.COM

Op  de B-drive  verschijnt nu  de file  SHOW-EXE.COM, een  file die we straks 
gewoon onder  DOS kunnen  aanroepen. Het voordeel hiervan is dat we dus files 
die  we vaak  gebruiken nu  gecrunched kunnen  opslaan, en  zonder eerst  met 
PMEXT.COM te moeten uitpakken kunnen gebruiken.

Iedereen die  goed heeft  opgelet weet dat je ook een *.COM via PMEXT.COM kan 
starten:

A>PMEXT SHOWEM.PMA EXE:SHOWEM.COM

Het  resultaat is dat SHOWEM.COM zonder te worden weggeschreven op disk wordt 
uitgepakt  en   gestart,  echter   SHOWEM.COM  heeft  wat  parameters  nodig, 
bijvoorbeeld de naam van een GIF-file (bijv. FREDDY.GIF).

Wanneer  we nu  de SHOW-EXE.COM  file aanroepen,  en gelijk de parameters van 
SHOWEM.COM (die  daar in  zit) meegeven  zal SHOWEM.COM  na uitpakken  ons de 
GIF-file laten zien:

A>SHOW-EXE FREDDY.GIF   --ฯ SHOWEM laat nu de GIF zien !!


Zoals je  misschien al  door hebt,  schept dit programma legio mogelijkheden.
