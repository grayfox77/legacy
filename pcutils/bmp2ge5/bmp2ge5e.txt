< How to Use BMP2GE5, the BMP -> SCREEN 5 converter >

- How to boot
Drag&drop a bitmap file (256x212 pixels, 16 colors) into
BMP2GE5.EXE itself or its shortcut.
For your convenience, place shortcut in Windows\SendTo\ folder,
then you can SEND a file to BMP2GE5 by right-clicking over a BMP
file.

- How to Convert
Boot BMP2GE5, then enter after-conversion file name to edit box,
then press Convert button. Or, cancel by Cancel button.
